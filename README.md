## Project Overview
This project performs a statistical audit of an open-source GitHub repository using concepts from:
- Parameter Estimation
- Confidence Intervals
- Hypothesis Testing
- Monte Carlo Simulation
- Bloom Filter
- Markov Chain Monte Carlo (MCMC)

## Repository Structure

data/clean/
Raw and cleaned datasets.

notebooks/
Jupyter notebooks for analysis.

src/
Python source modules.

presentation/
Final report and presentation materials.

## AI Usage
AI assistance was used for coding support and debugging. Statistical interpretations and hypotheses were written by team members.
